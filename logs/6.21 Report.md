近几年 embodied AI / robot learning 的主线可以理解为下面这条演化链。
`BC / Imitation Learning -> ACT / Diffusion Policy -> RT-1 / RT-2 -> OpenVLA / pi0 -> World Model`

| 路线                                    | 解决的问题                 | 和其他路线的关系                               |
| ------------------------------------- | --------------------- | -------------------------------------- |
| Behavior Cloning / Imitation Learning | 从专家示范中学习策略            | 底层训练范式                                 |
| ACT / Diffusion Policy                | 学习连续控制动作序列，缓解单步动作误差累积 | robot learning baseline                |
| VLA / Robot Foundation Model          | 用视觉、语言、动作统一建模，提升泛化    | 依赖大规模数据，也常结合 diffusion/action chunking |
| World Model / WAM                     | 预测未来状态、视频或动作-状态转移     | 目前最前沿的方向                               |

## Baseline
### Discrete action model
  - RT-2：早期代表，把 robot action 当作 text token，验证 web knowledge transfer 到机器人控制。
  - OpenVLA：开源 VLA baseline，7 B，OpenX 970 k episodes，open-source、fine-tuning 和 quantization。
  - WorldVLA：离散 autoregressive action world model，同时预测 action 和 future image。

decoder token -> action 代码

### Continuous action model：
  - Diffusion Policy：经典 diffusion-based imitation learning baseline，强在平滑连续动作和 action chunking。
  - Octo：OpenX 上训练的 open-source generalist robot policy，常作为 VLA 前的强通用 policy baseline。
  - DiT-Policy：diffusion transformer policy，在 LIBERO 中作为强连续动作 baseline。
  - π₀：VLM backbone + flow matching action expert，代表 continuous VLA / robot foundation model。
  - OpenVLA-OFT：OpenVLA 后续优化 fine-tuning 的强 baseline。
  - UVA：Unified Video Action Model，和 WorldVLA / DreamZero 一样属于 action + video/world modeling 方向。

## Dataset
- **Open X-Embodiment Dataset**：VLA 训练最重要的公开多机器人数据集合之一，OpenVLA 主要基于它训练。
- DROID / BridgeData V 2：常见真实机器人 manipulation 数据，用于 Franka、WidowX 等平台。
- **LIBERO：仿真 benchmark**，常用于评估 VLA / policy learning 的迁移与泛化。
- **RoboTwin 2.0**：双臂仿真数据生成器 + benchmark，重点是 domain randomization、bimanual manipulation 和 sim-to-real。

## Benchmark
- 实机 benchmark：RT-2、OpenVLA、π₀、DreamZero 的主要说服力来自 <mark style="background: #FF5582A6;">real-robot evaluation</mark>。
- 仿真 benchmark：WorldVLA 主要使用 **LIBERO**；**RoboTwin 2.0** 本身提供双臂仿真 benchmark；DreamZero 也补充 PolaRiS / Genie Sim 3.0。
